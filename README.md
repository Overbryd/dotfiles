# Overbryd’s dotfiles

## Installation

### Install tools and dotfiles

```bash
curl "tools-and-dotfiles.sh" | sh
```

### Install dotfiles only

```bash
curl "dotfiles.sh" | sh
```

### Setup sensible OS X defaults

When setting up a new Mac, you may want to set some sensible OS X defaults:

```bash
~/.osx
```
